\subsection{Description détaillée}
\hyperlink{class_twig___node___sandboxed_print}{Twig\+\_\+\+Node\+\_\+\+Sandboxed\+Print} adds a check for the \+\_\+\+\_\+to\+String() method when the variable is an object and the sandbox is activated.

When there is a simple Print statement, like \{\{ article \}\}, and if the sandbox is enabled, we need to check that the \+\_\+\+\_\+to\+String() method is allowed if \textquotesingle{}article\textquotesingle{} is an object.
